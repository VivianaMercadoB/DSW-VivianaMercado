Propósito del sitio

Diseñar y construir sitios web utilizando lenguaje HTML de acuerdo con los requerimientos del usuario. HTML es un lenguaje de marcación que sirve para definir el contenido de las páginas web y se compone en base a etiquetas.

Etiquetas Semánticas

Las etiquetas semánticas son etiquetas del lenguaje HTML que dan un significado al documento. Indican qué son y cuál es el significado del contenido que contienen dentro. A su vez, tienen mucha importancia en la composición de un documento y tienen múltiples funciones como separar un párrafo, poner títulos y subtítulos, enlaces, entre muchos otros. Según Fernández, A. G., & Vivar, I. U. (2006). Bibliografia: Fernández, A. G., & Vivar, I. U. (2006). Funciones textuales: etiquetas semánticas. In Linguistique plurielle: Valencia. 25, 26 et 27 Octobre 2006 (pp. 383-395). Departamento de Lingüística Aplicada.

CSS Grid

Las Css Grid básicamente contiene funciones de diseño, dirigidas a quienes desarrollan las aplicaciones y páginas web. Se puede utilizar para lograr muchos diseños diferentes. Destaca por dividir una página en regiones principales, o definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML. Según Fernández, A. G., & Vivar, I. U. (2006) Bibliografia: Aguirre, H.J.O. Tesina.http://repositorio.upsin.edu.mx/formatos/Tesina2595.pdf
